# TextTable

TextTable 是一个轻量的 PySide6 桌面应用，用于查看和处理使用自定义分隔符拆分的文本数据。

## 功能

- 拖拽文本文件或原始文本
- 使用分隔符 `----` 解析行
- 表格排序与多行选择
- 右键菜单：复制 / 删除 / 反选 / 全选
- 右键可切换“仅复制选中列”
- 过滤面板（全局 + 分列，AND 逻辑）
- 列管理与批量工具
- 去重 / 分组工具
- 导出 TXT / CSV / XLSX + 模板（导出时选择格式）
- 记住上次导出目录
- 启动自动恢复上次内容
- 主题切换与状态栏信息
- 撤销 / 重做

## 运行环境

- Python 3.8+（建议使用 venv）
- 依赖见 `requirements.txt`

## 安装

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

## 启动

```bash
python main.py
```

## 基本使用

1. 将文本文件拖入窗口，或直接粘贴原始文本。
2. 数据按 `----` 分隔并显示在表格中。
3. 右键表格可复制/删除/选择操作。
4. 使用过滤面板筛选数据。
5. 点击“导出”并选择格式。

## 快捷键

- Ctrl+C：复制选中行（列以 `----` 连接）
- Delete：删除选中行
- Ctrl+A：全选
- Ctrl+F：查找（上一个/下一个）
- Ctrl+Z：撤销
- Ctrl+Y：重做

## 说明

- 第一列为序号，不参与复制和导出。
- 点击表头可排序。

## 常见问题

- 如果出现中文乱码，应用会尝试 UTF-8 和常见中文编码（GBK/GB18030），
  可优先检查源文件编码。
